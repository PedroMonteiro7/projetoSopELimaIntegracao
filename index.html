Desenvolvimento de Sistemas

FPOO

O que é?

Ocupando 150 horas da carga horária do 1º semestre, a disciplina de Fundamentos de Programação Orientada a Objeto (FPOO) é a mais longa em relação as outras dessa primeira etapa e dá a base de como desenvolver um software, ou seja, é nesse componente curricular que se obtém uma noção de como fazer programas para computador, como, por exemplo, um gerenciador da movimentação de um estacionamento ou aqueles sistemas de cobrança, cálculo e registro que vemos nos supermercados.

Mas o que, especificamente, se aprende?

Como o seu próprio nome já diz, essa matéria é voltada aos fundamentos de programação orientada a objeto. Então, aprende-se a definição, os princípios, características e aplicação de linguagens de programação (somente Java no 1º semestre), lógica de programação orientada por cada objeto, do ambiente em si de desenvolvimento, da manipiulação de arquivos e do controle de versões.

LIMA

O que se aprende?

É nessa disciplina - Linguagem de Marcação - que se aprende a desenvolver sites/páginas Web. Sendo introduzido com definição, planejamento e semântica dos elementos, dá, através da prática, uma noção do quão gigante é esse mercado de trabalho e proporciona uma ótima base para quem pretende ser um desenvolvedor Web.

Mas qual é o foco exatamente?

O objetivo em si é justamente a criação de páginas Web (sites), utilizando a linguagem de marcação HTML e a estilização CSS e tendo como base de aprendizado os elementos da linguagem, versionamento de arquivos, os diferentes leiautes e disposição dos itens na tela, a manipulação de textos e imagens e a estilização destes.

SOP

O que é?

A disciplina de Sitemas Operacionais (SOP) trata justamente destes, que são um programa ou um conjunto de programas cuja função é gerenciar os recursos do sistema, fornecendo uma interface entre o dispositivo eletrônico e o usuário, isto é, o sistema operacional é o mediador entre a máquina e a pessoa utilizadora dela.

Definição e Histórico

Nessa introdução, o(a) aluno(a) entende o que é um sistema operacional, seus conceitos, funcionamento e vê o histórico, a evolução e a competição entre as gigantes empresas tecnológicas nesse processo e até os dias de hoje.

Instalação

Aqui se aprende como instalar um sistema operacional através de simulações, mas que fornecem todos os conceitos e fundamentos necessários. Instala-se o Windows XP, Linux Ubuntu, Debian e o Windows 10, que está exemplificado.

Programas Básicos

E por fim, já tendo instalado todos os sistemas mencionados, se conhece os programas e funcionalidades básicss de cada sistema operacional, como editores de texto, planilhas de cálculo, slides e sistema de arquivos.

HARE

Introdução

A disciplina de Hardware e Redes é introduzida com conceitos básicos e, principalmente, com a abordagem de sistemas de numeração de diferentes bases, como octal, hexadecimal e, em especial, binário, os quais são essenciais durante todo o componente curricular.

Redes

O cursando obtém uma base do que seria o curso de Redes, o que é imprescindível para um(a) programador(a). Ele aprende coisas como definição, aplicabilidade, tipos, topologia, protocolo, endereçamento IP, meios físicos e sem fio de transmissão de dados e unidades e pastas em rede.

Hardware

Hardware é toda a parte física do computador, como, por exemplo: placa-mãe, processador, memória RAM, fonte de alimentação, disco rígido, etc. Então, é aqui que se instala e configura, de fato, um sistema operacional, drivers e aplicativos periféricos em uma máquina real e ainda se realiza testes de funcionamento.
